# test-thv-in-codespaces

This repository contains a demo application showcasing a slightly complex hello-world application.

## Demo App

The `demo_app` directory contains a Python CLI application managed with `uv` and built with `click`.

### Quick Start

```bash
cd demo_app
uv sync
uv run demo-app --help
```

See [demo_app/README.md](demo_app/README.md) for detailed documentation.
